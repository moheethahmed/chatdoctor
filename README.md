# 🩺 ChatDoctor AI

ChatDoctor is an AI-powered medical assistant that predicts diseases based on symptoms and suggests home remedies.

## 🚀 Features
- Select symptoms from a list
- Predicts possible condition
- Shows remedy
- 🎤 Speaks the diagnosis out loud
